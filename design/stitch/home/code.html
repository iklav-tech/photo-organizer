A powerful Python CLI tool to organize local photo collections by renaming and sorting into directories based on metadata, date, time, and fully configurable custom rules.

Quick Start

terminal — bash 
# Create and activate environment
$ python3 -m venv .venv
$ source .venv/bin/activate

# Install the CLI
$ pip install photo-organizer

# Audit your current collection
$ photo-organizer scan ./Photos

# Organize with a dry-run first
$ photo-organizer organize ./Photos --output ./Sorted --dry-run
