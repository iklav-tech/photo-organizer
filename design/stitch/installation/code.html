Instalação

Configure seu ambiente e prepare-se para organizar sua biblioteca de fotos com precisão cirúrgica e alta performance.

checklist

Requisitos

Runtime

  • check_circle Python >= 3.10
  • check_circle pip e venv instalados

Dependências

  • check_circle Pillow & PyYAML
  • check_circle pillow-heif (opcional)
info

Para suporte HEIC/HEIF, o backend nativo libheif pode ser necessário se o wheel local não trouxer suporte nativo suficiente.

terminal

Instalação para desenvolvimento

Ambiente Virtual

Recomendamos o uso de ambientes isolados para evitar conflitos de bibliotecas.

python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"

Instalação Simples

Alternativa utilizando o arquivo de requerimentos clássico:

pip install -r requirements.txt
verified

Verificar a CLI

zsh — photo-organizer
$ photo-organizer scan ./Photos
$ photo-organizer inspect ./Photos --report metadata-audit.json
$ photo-organizer organize ./Photos --output ./OrganizedPhotos --dry-run
science

Executar testes

Garanta a integridade do seu setup rodando a suíte de testes integrada com pytest.

$ pytest

Nota: Alguns testes de HEIC podem ser pulados se o ambiente local não suportar gravação via libheif.
