Inicio / Instalacao / Uso

Uso

Aprenda a utilizar a ferramenta de linha de comando photo-organizer para escanear, auditar e organizar sua biblioteca de fotos com precisão técnica.

Comando base

bash
photo-organizer [--version] [--log-level DEBUG|INFO|WARNING|ERROR|CRITICAL] <comando>

scan

lista imagens suportadas em uma pasta.

dedupe

identifica duplicatas por hash de conteudo; somente leitura.

inspect

audita metadados disponiveis e decisoes finais.

organize

move fotos para uma estrutura organizada.

folder_open

scan

photo-organizer scan ./Photos

Formatos suportados

.arw .cr3 .dng .heic .jpeg .nef .raf .tiff .webp
content_copy

dedupe

photo-organizer dedupe ./Photos photo-organizer dedupe ./Photos --report duplicates.json photo-organizer dedupe ./Photos --read-only
info

O comando dedupe não move, copia nem remove arquivos. O relatório aceita formatos .json ou .csv.

search_check

inspect

photo-organizer inspect ./Photos photo-organizer inspect ./Photos --report metadata-audit.csv
Opção Descrição
--report PATH Grava auditoria em .json ou .csv.
--clock-offset OFFSET Ajuste de fuso horário: +3h, -1d, etc.
--reverse-geocode Resolve coordenadas em endereços.
architecture

organize & import

photo-organizer organize ./Photos --output ./Organized --copy

Por padrão, o comando organize move os arquivos. Use --copy para manter a origem intacta. O import faz o inverso: copia por padrão e só move se explicitado com --move.

calendar_month

Agrupamento

Organize por data, evento, local ou combinação via --by.

auto_fix_high

Burst

Detecção inteligente de disparos contínuos via --burst-detection.
